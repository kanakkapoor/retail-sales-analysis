# 🛒 Retail Sales Analysis Using SQL (MySQL)

## Overview

This project demonstrates the practical application of SQL for analyzing retail sales data and generating business insights. By working with a relational database consisting of Customers, Products, and Orders tables, I explored customer behavior, product performance, category-wise sales, and overall profitability.

The goal of this project was to strengthen my SQL skills while understanding how data can be used to support business decision-making.

---

## Project Objectives

* Analyze overall sales and profit.
* Identify top-performing customers.
* Evaluate product and category performance.
* Calculate key business metrics using SQL.
* Practice real-world SQL queries on a relational database.

---

## Database Structure

The project consists of three related tables:

### Customers

* Customer ID
* Customer Name
* City
* State
* Segment

### Products

* Product ID
* Product Name
* Category
* Sub-Category
* Price

### Orders

* Order ID
* Order Date
* Customer ID
* Product ID
* Quantity
* Sales
* Profit
